# Requirements
1. ruby 2.7
2. node 12
3. yarn

# Development Setup
1. `$ bin/setup`
2. `$ overcommit --install`
3. `$ overcommit --sign`
4. `$ overcommit --sign pre-commit`

# JS Testing
Use Jest for javascript testing
`$ yarn test`

# Ruby Testing
Use Rspec for ruby/rails testing
`$ bundle exec rspec`
